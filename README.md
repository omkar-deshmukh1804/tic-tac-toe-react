## Run Locally

Clone the project

```bash
  git clone https://github.com/omkar-deshmukh1804/tic-tac-toe-react
```

Go to the project directory

```bash
  cd tic-tac-toe-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
